# AGENTS.md

## Project

Spanish civil service exam simulator (GVA 84/26 - Ayudante de Cocina). Single-page web app using vanilla HTML/CSS/JS.

## Files

- `index.html` - Application UI and logic
- `exam.json` - Question bank (10 general + 20 specific)

## Changes

- Edit questions in `exam.json` following the existing schema
- Modify UI in `index.html` (inline CSS in `<style>`, JS in `<script>`)

## Running

Open `index.html` directly in a browser. No build or server needed.
