# Objetivo

Modificar el código para que satisfaga el siguiente requisito.

- Configuración del examen:
  - Estructura: 60 preguntas (28 generales / 32 específicas).
  - Tiempo: 60 minutos.
  - Cálculo: Penalización de -0,33 (1/3) por fallo.
  - Contenido: Incremento de dificultad en el área de cocina.

Haz que esos parámetros sean configurables y que por defecto tengan los valores que pide el nuevo requisito.
